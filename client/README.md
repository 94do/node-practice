## 命名规范

- class 用短横线
- 变量 用驼峰
- 常量 全大写
- 请求接口的方法用api_*
- 普通方法用fn_*
- 计算属性用cpt_*
- 传递的参数用data* 驼峰
- 全平台通用方法 用驼峰 放在common文件夹下
